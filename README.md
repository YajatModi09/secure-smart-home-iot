# 🏠 Secure Smart Home IoT System

## 📌 Overview

This project demonstrates a **secure Smart Home IoT system** implementing core **Information Security principles** such as authentication, encryption, and data integrity.

It simulates how IoT devices (like smart locks and sensors) securely communicate with users.

---

## 🚀 Features

* 🔐 User Authentication (SHA-256 hashed passwords)
* 🔒 AES Encryption (using Fernet)
* 🧾 Data Integrity Check (Hash verification)
* 🛡️ Role-Based Access Control
* 📡 Secure Command Transmission

---

## 🧠 Security Concepts Covered

| Concept         | Implementation                     |
| --------------- | ---------------------------------- |
| Authentication  | Login system with hashed passwords |
| Confidentiality | AES encryption                     |
| Integrity       | SHA-256 hashing                    |
| Authorization   | Role-based access                  |

## ⚙️ Installation & Setup

### 1. Clone Repository

```bash
git clone https://github.com/YajatModi09/secure-smart-home-iot.git
cd secure-smart-home-iot
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the Project

```bash
python src/main.py
```
